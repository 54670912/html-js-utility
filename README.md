## 📘 Sobre / About  

🇧🇷 **HTML JS Utility** é uma biblioteca educacional desenvolvida para ajudar iniciantes a praticar programação, fortalecer a lógica e entender como HTML e JavaScript se conectam de forma simples e eficaz.  

🇺🇸 **HTML JS Utility** is an educational library built to help beginners practice coding, strengthen logic, and understand how HTML and JavaScript connect in a simple and efficient way.
